Mirror of FS23 iteration of ETH Zurich lecture _Deep Learning for Scientific Computing_

```bash
micromamba create -n dlsc pytorch torchvision scikit-learn numpy matplotlib juypter
micromamba activate dlsc
jupyter notebook
```
